At least recently, there haven't been many accordions created as part of UI libraries for React components, so I created my own. Vertical Rhythm with CSS

I created a CSS drop-in to kick off my projects with better vertical rhythm. In conventional CSS modeling and browser rendering there's no real concept for vertical rhythm and a bunch of browser defaults and quirks throw things off. This helps to get things back on track. Because its CSS only it doesn't account for images which can still shift elements out of rhythm.

Github Link

See the Pen Vertical Rhythm with CSS by austin (@austinnnnnnn) on CodePen.
